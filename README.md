# Multi-Threaded-Client-Server

## Prerequisites
 - Running LAMMP (or equuivilant) server
 - Populated Assign1 table with following spec : 
      - SID (type int(2) , unique key)
      - STUD_ID (type int(8))
      - FNAME (type varchar(20))
      - SNAME (type varchar(20))
      - TOT_REQ (type int(8))
